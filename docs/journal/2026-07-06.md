## Work log

### CP1 — schema, service, and payment hooks

**Migration** (`packages/db/prisma/migrations/20260705000000_phase_3_slice_9_payment_plans/migration.sql`):

- `payment_plans` table: `id`, `school_id`, `invoice_id`, `name`, `created_by`, `created_at`.
  Unique index on `(school_id, invoice_id)` — one plan per invoice.
- `payment_plan_installments` table: `id`, `school_id`, `plan_id`, `amount` (INTEGER kobo),
  `due_date` (DATE), `paid` (BOOLEAN DEFAULT false).
- ENABLE/FORCE RLS + `tenant_isolation` policy on both tables, matching the
  pattern used by all other Phase 3 finance tables.

**`PaymentPlanService`** (`apps/api/src/modules/payments/payment-plan.service.ts`):

- `computeInstallmentsPaid(amounts, totalPaid)` — exported pure helper.
  Walks cumulative sums chronologically; `paid[i] = cumulative[i] <= totalPaid`.
- `create(authCtx, dto)` — enforces D2 (sum constraint), D3 (one plan per invoice),
  D4 (ISSUED or PARTIALLY_PAID only), then immediately calls `recomputeInstallmentsPaid`
  to auto-mark installments paid on PARTIALLY_PAID invoices.
- `findByInvoice(authCtx, invoiceId)` — returns `PaymentPlanDto | null`.
- `delete(authCtx, planId)` — blocked with `PLAN_LOCKED_PAYMENTS_EXIST` if
  `invoice.totalPaid > 0`.
- `recomputeInstallmentsPaid(db, invoiceId, totalPaid)` — accepts the open
  tenant-scoped `PrismaClient` from the caller's `withTenant` callback to avoid
  a second connection. No-op if no plan exists for the invoice.

**Spec** (`apps/api/src/modules/payments/payment-plan.service.spec.ts`):
17 tests — 8 pure unit (`computeInstallmentsPaid`) + 9 integration. All green.

**`PaymentsService` hooks** (`apps/api/src/modules/payments/payments.service.ts`):

- `recomputeInstallmentsPaid` called at the end of both `recordManual` (step 9)

**Supporting changes**:

### CP2 — controller, API client, invoice UI

**`PaymentPlansController`** (`apps/api/src/modules/payments/payment-plans.controller.ts`):

```
GET    /payment-plans/invoice/:invoiceId   @Permissions('payment-plan.read')
POST   /payment-plans                      @Permissions('payment-plan.create')
DELETE /payment-plans/:id                  @Permissions('payment-plan.delete')
```

Static sub-path `invoice/:invoiceId` declared before `/:id` to prevent NestJS
from capturing the literal string `"invoice"` as an id parameter.
